================================================================================
HOW IT WORKS
================================================================================

Pipeline:
  1. NewsNow API → Get article URL
  2. Content Extractor → Extract full text (using Mozilla Readability)
  3. AI Cleaner → Summarize content (2-3 sentences)
  4. Translator → Translate to EN/ZH/MS

Example:
  Input:  "U.S. rig counts remain low..."
  URL:    https://www.eia.gov/todayinenergy/detail.php?id=66645
  
  Extract: 2778 characters of article content
  
  Summary: "U.S. Lower 48 active rigs remained at low levels in 
           October 2025, with 517 rigs drilling, down from 750 in 
           December 2022, as operators pursue efficiency gains..."
  
  Translate:
    EN: U.S. rig counts remain low as production efficiencies improve
    ZH: 美国钻井数量保持低位，生产效率提高
    MS: Kiraan pelantar AS kekal rendah ketika kecekapan...

================================================================================
USAGE
================================================================================

BASIC (Title Only - Backward Compatible):
  from ai_processing import ArticleProcessor
  processor = ArticleProcessor(...)

ENHANCED (With Content Extraction):
  from ai_processing import ArticleProcessorWithContent
  
  processor = ArticleProcessorWithContent(
      api_url="https://api.bltcy.ai/v1/",
      api_key="your-key",
      model="gpt-5-nano-2025-08-07",
      extract_content=True  # Enable content extraction
  )
  
  processed = processor.process_single(raw_article)
  
  # Access results
  print(processed.metadata['content'])   # Full article
  print(processed.metadata['summary'])   # AI summary
  print(processed.title_en)              # English
  print(processed.title_zh)              # Chinese
  print(processed.title_ms)              # Malay

================================================================================
NEW COMPONENTS
================================================================================

1. ContentExtractor (services/content_extractor.py)
   - Uses Mozilla Readability algorithm
   - Extracts clean article text from HTML
   - Removes ads, navigation, footers

2. ContentCleaner (services/content_cleaner.py)
   - AI-powered summarization
   - Removes promotional content
   - Creates 2-3 sentence summaries

3. ArticleProcessorWithContent (processor_with_content.py)
   - Enhanced processor with content extraction
   - Orchestrates full pipeline
   - Backward compatible

================================================================================
ERROR HANDLING
================================================================================

Graceful Fallbacks:
  1. Content extraction fails → Use title only
  2. AI summarization fails → Return original title
  3. Translation fails → Return summary in all languages

No crashes, always returns results!
